# 데이터 Import 가이드

## 📋 Import 순서

### 1단계: 제품스펙 Import (완료 ✅)
```bash
python manage.py import_all_categories
```
**현재 상태**: 1,420개 제품, 1,419개 스펙 완료

---

### 2단계: 인구통계, 추천이유, 리뷰 Import
```bash
python manage.py import_reviews_demographics
```

**현재 상태**:
- ✅ 인구통계: 568개 완료
- ✅ 추천이유: 224개 완료
- 🔄 리뷰: 95,447개 / 130,000개 (73% 진행중)

**예상 소요 시간**: 리뷰 import는 13만개+ 처리하므로 10~30분 정도 걸릴 수 있습니다.

---

## 📊 진행 상황 확인

```bash
python manage.py check_data
```

이 명령어로 현재까지 import된 데이터 수를 확인할 수 있습니다.

---

## 💡 팁

1. **백그라운드 실행**: 
   - 리뷰 import는 시간이 오래 걸리므로, 점심 먹으러 가기 전에 실행해두시면 됩니다.
   - 터미널을 닫지 않으면 계속 진행됩니다.

2. **중단해도 괜찮음**:
   - 이미 import된 데이터는 유지됩니다.
   - 나중에 다시 실행하면 남은 부분만 계속 진행됩니다.

3. **완료 확인**:
   - 점심 먹고 돌아와서 `python manage.py check_data` 실행
   - 리뷰가 130,000개 이상이면 완료!

---
