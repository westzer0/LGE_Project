# 취향별 Scoring Logic 설명서

총 50개의 독립적인 Scoring Logic이 생성되었습니다.

## 목차

1. [개요](#개요)
2. [Logic 구조](#logic-구조)
3. [각 Logic별 상세 설명](#각-logic별-상세-설명)

## 개요

768개의 취향 조합을 유사한 그룹으로 묶어 50개의 독립적인 Scoring Logic을 생성했습니다.

각 Logic은 해당 취향 그룹의 특성을 반영하여 제품 추천 점수를 계산합니다.

## Logic 구조

각 Scoring Logic은 다음 요소로 구성됩니다:

- **weights**: 카테고리별 속성 가중치
- **filters**: 필터링 규칙 (must_have, should_have, exclude)
- **bonuses**: 보너스 점수 규칙
- **penalties**: 페널티 점수 규칙
- **explanation**: Logic 설명
- **rationale**: 도출 근거
