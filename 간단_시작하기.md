# 🚀 3단계로 시작하기 (5분 완성!)

## 1단계: ngrok 가입 (2분)
1. https://ngrok.com 접속
2. "Sign up" 클릭 (GitHub로 가입하면 빠름)
3. 가입 완료 후 https://dashboard.ngrok.com/get-started/your-authtoken 접속
4. **"Your Authtoken" 복사** (예: `2abc123def456ghi789jkl012mno345pq`)

## 2단계: 자동 설정 (1분)
```powershell
powershell -ExecutionPolicy Bypass -File setup_ngrok.ps1
```
토큰 입력하라고 나오면 1단계에서 복사한 토큰 붙여넣기

## 3단계: 서버 시작 (1분)
```cmd
start_server.bat
```

끝! ngrok 창에서 나온 URL을 카카오 개발자 콘솔에 등록하세요!

---

## 🆘 문제 해결

### "ngrok을 찾을 수 없습니다" 오류
→ 2단계를 다시 실행하세요

### "인증 토큰 설정 실패"
→ 토큰을 다시 확인하고 입력하세요

### 포트 8000이 이미 사용 중
→ `start_server.bat` 파일을 열어서 `8000`을 다른 포트로 변경 (예: `8080`)
