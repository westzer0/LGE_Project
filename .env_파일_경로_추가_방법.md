# .env 파일에 경로 추가하는 방법 (가장 간단한 방법)

## 🎯 가장 쉬운 방법 (3단계)

### 1단계: 메모장으로 .env 파일 열기

**PowerShell에서** (현재 프로젝트 폴더에서):
```powershell
notepad .env
```

이 명령이 `.env` 파일을 메모장으로 열어줍니다!

---

### 2단계: 파일 끝에 한 줄 추가

메모장이 열리면, 파일의 **맨 끝**에 다음 줄을 추가하세요:

```
ORACLE_INSTANT_CLIENT_PATH=C:\oraclexe\instantclient-basic-windows.x64-23.26.0.0.0
```

**예시** (파일이 이렇게 생겼다면):
``` 
ORACLE_USER=campus_24K_LG3_DX7_p3_4
ORACLE_PASSWORD=smhrd4
ORACLE_HOST=project-db-campus.smhrd.com
ORACLE_PORT=1524
ORACLE_SID=xe
ORACLE_INSTANT_CLIENT_PATH=C:\oraclexe\instantclient-basic-windows.x64-23.26.0.0.0
```

**마지막 줄**이 추가되면 됩니다!

---

### 3단계: 저장하기

**Ctrl + S** 키를 누르거나 메뉴에서 **파일 → 저장** 클릭

끝! ✅

## 🔍 확인하기

저장한 후 PowerShell에서:

```powershell
python -c "import os; from dotenv import load_dotenv; from pathlib import Path; load_dotenv(Path('.env')); print('✅ 경로:', os.getenv('ORACLE_INSTANT_CLIENT_PATH'))"
```

이 명령이 경로를 출력하면 성공입니다!

---

## ⚠️ 주의사항

### 올바른 형식:
```
ORACLE_INSTANT_CLIENT_PATH=C:\oraclexe\instantclient-basic-windows.x64-23.26.0.0.0
```

### 잘못된 형식들:
```
ORACLE_INSTANT_CLIENT_PATH = C:\...  ← 공백 있음 (X)
"ORACLE_INSTANT_CLIENT_PATH=C:\..."  ← 따옴표 있음 (X)
# ORACLE_INSTANT_CLIENT_PATH=...     ← 주석 처리됨 (X)
```

## 🆘 .env 파일이 없어요

`.env` 파일이 없다면 먼저 만들어야 합니다:

```powershell
Copy-Item env.example .env
notepad .env
```

그 다음 위의 2-3단계 진행!

---

## 💡 한 번에 추가하는 방법 (고급)

PowerShell에서 한 번에 추가:

```powershell
Add-Content -Path .env -Value "`nORACLE_INSTANT_CLIENT_PATH=C:\oraclexe\instantclient-basic-windows.x64-23.26.0.0.0"
```

이 명령이 파일 끝에 자동으로 추가해줍니다!
