## 3. 추천 엔진 테스트

  - [ ] Lifestyle Score 계산이 정확한가?
  - [ ] Review Score 계산이 정확한가?
  - [ ] Price Score 계산이 정확한가?
  - [ ] Total Score 계산이 정확한가?

### 3.2 제품 타입별 추천
- [ ] 필수 제품 타입 추천이 정상적인가? (TV, 냉장고, 에어컨, 세탁기)
- [ ] 조건부 제품 타입 추천이 정상적인가?
  - [ ] 가구원수 기반 (4인 이상 → 건조기, 식기세척기)
  - [ ] 평수 기반 (35평 이상 → 공기청정기, 제습기)
  - [ ] 가격대 기반 (고가 → 와인셀러, 안마의자)
  - [ ] 온보딩 데이터 기반 (요리 빈도 → 식기세척기, 전기레인지)
- [ ] 제품 타입별 최대 3개 추천이 정상적인가?
- [ ] 중복 제품 제거가 정상적으로 작동하는가?

### 3.3 추천 결과 검증
- [ ] 추천 결과가 사용자 프로필과 일치하는가?
- [ ] 추천 결과에 필수 정보가 포함되어 있는가?
  - [ ] product_id
  - [ ] name/model
  - [ ] category
  - [ ] price
  - [ ] image_url
  - [ ] total_score
  - [ ] score_breakdown
  - [ ] explanation
- [ ] 추천 결과가 적절한 개수인가? (제품 타입별 3개씩)
- [ ] 추천 결과 순위가 점수 순서대로 정렬되어 있는가?

### 3.4 추천 API 테스트
- [ ] `POST /api/recommend/` 엔드포인트가 정상 작동하는가?
- [ ] 잘못된 입력 데이터 시 에러 처리가 되는가?
- [ ] 추천 결과가 JSON 형식으로 반환되는가?
- [ ] 응답 시간이 적절한가? (3초 이내 권장)
