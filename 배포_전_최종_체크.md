### 2. .env 파일이 Git에 추가되지 않았는지 확인

```bash
git status
# .env 파일이 목록에 나타나지 않아야 합니다!
```

만약 `.env` 파일이 나타난다면:
```bash
git rm --cached .env  # Git 추적에서만 제거
```

### 3. settings.py에 하드코딩된 비밀번호 확인

다음 명령어로 확인:
```bash
# Windows PowerShell
Select-String -Path "config\settings.py" -Pattern "password|secret|api_key" -CaseSensitive:$false
```

하드코딩된 비밀번호가 있으면 안 됩니다!
